
⚡ Energy Calculator (E = mc²)


📌 Overview
This project is a simple web-based energy calculator that computes energy using Einstein’s formula E = mc².
It also includes a fun mode where energy is represented using relatable concepts like milk and chocolate, along with dynamic image changes based on the result.


🎯 Features
Calculate energy using E = mc²
Simple and clean user interface
Takes user input (mass)
Displays calculated energy instantly
Fun mode: converts energy into milk 🍶 and chocolate 🍫 units
Dynamic image changes based on energy level


🧠 Formula Used
E = m * c²
Where:
E = Energy
m = Mass
c = Speed of light (3 × 10⁸ m/s)


🛠️ Tech Stack
HTML
CSS
JavaScript


▶️ How to Use
Enter mass value in the input field
Click on Calculate
View the energy result
See fun conversion (milk/chocolate)
Image changes based on energy level


🎨 UI Features
Minimal and user-friendly design
Interactive result display
Visual feedback using images
